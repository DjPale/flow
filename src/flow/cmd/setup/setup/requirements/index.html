home

Some platforms have post requirements, which are listed on the index page as step two.
If you already have any of the requirements installed, you can skip to the next step.


mac and ios target requirements


Xcode

Xcode is a free development environment provided by Apple for Mac.
You need version 4 or 5 installed as well as the command line tools.


linux target requirements


GCC / G++

To build native applications on linux you need a gcc/g++ install using the method that best suites your distro.

distro specific suggestions

Ubuntu

apt-get can be used to install the requirements using the following commands in the terminal

Ubuntu 12~14
sudo apt-get install gcc-multilib g++-multilib libxext-dev



windows target requirements


Visual C++

Visual C++ is a development environment provided by Microsoft for Windows.
You need 2010 C++ or higher, the link above is for Visual C++ Express 2010.


android target requirements


Android SDK

You only need the SDK Tools, as long as you know the path to the SDK.
You need to manually install target platforms you want to target,
documented in the Android SDK, available by running the android sdk manager.

Android NDK

Since android applications are often built as native applications, the NDK install is a good idea.



web target requirements

When building against the web target, there is no setup required.

You can skip to the next step.
